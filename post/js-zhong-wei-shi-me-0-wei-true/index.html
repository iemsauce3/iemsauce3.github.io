JS中为什么(' ' == 0) 为true

· 2019-02-15 · # 前端 # js

两边类型不相同出现以下情况：
0为假即false
空值也或空格也为false
false==false恒成立

只有两边类型相同时才可以真正对比是否完全一样
比如
"a"=="b"返回false
1==2返回false

解决方案：

可以用 isNaN() 判断是不是 或 typeOf() 判断好类型再去判断 parseInt() 转int 值等
